# Tiket Kapal Penyebrangan Kayangan - Pototano

Website promosi dan pemesanan tiket kapal penyebrangan.

## Cara Menggunakan

1.  Buka folder ini di File Explorer.
2.  Klik dua kali pada file `index.html`.
3.  Website akan terbuka di browser default Anda.

## Fitur

*   **Pemesanan Tiket 4 Langkah**: Jadwal, Jenis Tiket, Data Diri, Pembayaran.
*   **Kalkulasi Harga Otomatis**: Harga total dihitung secara real-time berdasarkan jumlah penumpang dan kendaraan.
*   **Responsive Design**: Tampilan optimal di PC, Tablet, dan Smartphone.
*   **Informasi Lengkap**: Keunggulan, FAQ, dan Kontak.

## Teknologi

*   HTML5
*   CSS3 (Modern, dengan variabel dan flexbox/grid)
*   JavaScript (Vanilla)
